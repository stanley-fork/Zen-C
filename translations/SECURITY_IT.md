# Politica di Sicurezza

## Versioni Supportate

Usa questa sezione per indicare quali versioni del tuo progetto ricevono aggiornamenti di sicurezza.

| Versione | Supportata         |
| -------- | ------------------ |
| 0.1.x    | :white_check_mark: |
| < 0.1    | :x:                |

## Segnalare una Vulnerabilità

Prendiamo molto seriamente la sicurezza di Zen C. Se credi di aver trovato una vulnerabilità di sicurezza nel compilatore Zen C, nella libreria standard o negli strumenti associati, ti preghiamo di segnalarcelo come descritto di seguito.

**Per favore, non segnalare vulnerabilità di sicurezza tramite issue pubbliche su GitHub.**

### Come Segnalare

Per favore inviaci un'email a **security@zenc-lang.org**.

Nella tua segnalazione, per favore includi:

- La versione di Zen C che stai utilizzando (`zc --version`).
- Una descrizione della vulnerabilità.
- Un Esempio Minimo Riproducibile (MRE) o passaggi per riprodurre il problema.
- Qualsiasi configurazione o dettaglio dell'ambiente potenzialmente rilevante.

Dovresti ricevere una risposta entro 48 ore. Se il problema viene confermato, rilasceremo una patch il prima possibile.
