# 安全政策

## 支援的版本

請使用本節了解哪些版本的專案目前正在接收安全更新。

| 版本    | 支援               |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## 報告漏洞

我們非常重視 Zen C 的安全性。如果您認為在 Zen C 編譯器、標準庫或相關工具中發現了安全漏洞，請按照下面的描述向我們報告。

**請不要透過公開的 GitHub Issue 報告安全漏洞。**

### 如何報告

請發送電子郵件至 **security@zenc-lang.org**。

在您的報告中，請包括：

- 您正在使用的 Zen C 版本 (`zc --version`)。
- 漏洞的描述。
- 最小可複現範例 (MRE) 或複現問題的步驟。
- 任何可能相關的配置或環境細節。

您應該在 48 小時內收到回覆。如果問題得到確認，我們將盡快發布修復補丁。
