| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reportando uma Vulnerabilidade

Levamos a segurança do Zen C muito a sério. Se você acredita que encontrou uma vulnerabilidade de segurança no compilador Zen C, na biblioteca padrão ou nas ferramentas associadas, relate-nos conforme descrito abaixo.

**Por favor, não relate vulnerabilidades de segurança através de issues públicas no GitHub.**

### Como Reportar

Por favor, envie um e-mail para **security@zenc-lang.org**.

Em seu relatório, por favor inclua:

- A versão do Zen C que você está usando (`zc --version`).
- Uma descrição da vulnerabilidade.
- Um Exemplo Mínimo Reproduzível (MRE) ou passos para reproduzir o problema.
- Qualquer configuração ou detalhe do ambiente potencialmente relevante.

Você deve receber uma resposta dentro de 48 horas. Se o problema for confirmado, lançaremos uma correção o mais rápido possível.
