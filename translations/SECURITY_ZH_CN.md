# 安全政策

## 支持的版本

请使用本节了解哪些版本的项目目前正在接收安全更新。

| 版本    | 支持               |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## 报告漏洞

我们非常重视 Zen C 的安全性。如果您认为在 Zen C 编译器、标准库或相关工具中发现了安全漏洞，请按照下面的描述向我们报告。

**请不要通过公开的 GitHub Issue 报告安全漏洞。**

### 如何报告

请发送电子邮件至 **security@zenc-lang.org**。

在您的报告中，请包括：

- 您正在使用的 Zen C 版本 (`zc --version`)。
- 漏洞的描述。
- 最小可复现示例 (MRE) 或复现问题的步骤。
- 任何可能相关的配置或环境细节。

您应该在 48 小时内收到回复。如果问题得到确认，我们将尽快发布修复补丁。
