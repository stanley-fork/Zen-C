## Reporting a Vulnerability

We take the security of Zen C seriously. If you believe you have found a security vulnerability in the Zen C compiler, standard library, or associated tooling, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please email us at **security@zenc-lang.org**.

In your report, please include:

- The version of Zen C you are using (`zc --version`).
- A description of the vulnerability.
- A Minimal Reproducible Example (MRE) or steps to reproduce the issue.
- Any potentially relevant configuration or environment details.

You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible.
